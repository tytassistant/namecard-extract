Connect Microsoft
  1. Go to Azure Portal → App registrations → your existing app (or create one)
  2. Under Authentication → Add a platform → Web
  3. Set Redirect URI to: http://localhost:8001/auth/ms/callback
  4. Copy the Application (client) ID
  5. Put it in helpers/.env → MS_CLIENT_ID=your_id_here

Connect Google
  1. Go to Google Cloud Console → APIs & Services → Credentials
  2. Create (or use existing) OAuth 2.0 Client ID → type: Web application
  3. Add Authorized redirect URI: http://localhost:8001/auth/google/callback
  4. Copy the Client ID and Client Secret
  5. Put them in helpers/.env → GOOGLE_CLIENT_ID and GOOGLE_CLIENT_SECRET
  6. Make sure the People API is enabled

Note: You need a Web application credential type (not Desktop) because the server needs a client secret to exchange the auth code.

OAuth setup only needs to be done once from this machine (localhost). Tokens are saved server-side. After connecting, any device on your network can sync contacts without OAuth.
